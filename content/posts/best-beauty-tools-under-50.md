## Beauty Tool Safety Guidelines

**Heat Styling Tools:**
- Always use heat protectant products
- Start with lower temperatures and increase as needed
- Never use on wet hair unless specified as safe
- Allow tools to cool completely before storing

**Skincare Devices:**
- Start with lowest intensity settings
- Perform patch tests before full-face use
- Clean devices thoroughly after each use
- Replace attachments as recommended by manufacturer

**Hair Removal Tools:**
- Read instructions completely before first use
- Test on small area first
- Exfoliate 24 hours before use
- Moisturize regularly between treatments
